# LinkedIn Locations Collector Extension (Demo)

Расширение Chrome для автоматического сбора и экспорта локаций LinkedIn-контактов.

**Создано для быстрого прототипирования и тестирования гипотез с помощью инструментов искусственного интеллекта.**

---

## Возможности

- Автоматический сбор имён и местоположений профилей LinkedIn при просмотре списка контактов.
- Сохранение данных в локальное хранилище браузера (chrome.storage.local).
- Включение/выключение авто-сбора через popup.
- Экспорт собранных данных в CSV одним кликом.
- Сброс данных и состояния сбора.
- Минималистичный и быстрый интерфейс popup.
- Подробное логирование для отладки (logInfo).

---

## Быстрый старт

1. Откройте chrome://extensions и включите "Режим разработчика".
2. Нажмите "Загрузить распакованное расширение" и выберите папку `linkedin-locations-collector extention`.
3. Откройте LinkedIn → Мои контакты.
4. Включите авто-сбор через popup расширения.
5. Данные будут собираться автоматически при прокрутке/переходах.
6. Для экспорта нажмите "Экспорт" в popup — получите CSV-файл.

## Структура проекта

- `manifest.json` — описание расширения и разрешения.
- `content.js` — сбор данных с LinkedIn, логика автообновления, логирование.
- `popup.js` — UI popup, экспорт, сброс, управление сбором.
- `background.js` — обработка скачивания CSV.
- `modules/`
  - `logger.js` — централизованный логгер.
  - `utils.js` — функции для работы с DOM, формирование CSV.
  - `constants.js` — id, сообщения, ключи storage.
- `popup.html` — интерфейс расширения.
- `block-example.html` — пример страницы для теста.
- `icon*.png` — иконки расширения.
